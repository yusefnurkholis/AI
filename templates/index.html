Materi Proyek

Alur pengembangan model

Isi materi mengikuti delapan tahap pada notebook Google Colab.

01

Data Collection

Dataset Rice Leaf Disease Images diunduh dari Kaggle melalui kagglehub.

02

EDA

Distribusi kelas, sampel gambar, resolusi, gambar rusak, dan duplikat diperiksa.

03

Data Processing

Data dibagi 70:15:15, di-resize, di-batch, dan diaugmentasi.

04

Feature Engineering

EfficientNetB0 mengekstraksi pola warna, tekstur, bercak, dan bentuk kerusakan.

05

Modelling

Transfer learning dilanjutkan fine-tuning layer terakhir dengan learning rate kecil.

06

Evaluation

Accuracy, loss, precision, recall, F1-score, dan confusion matrix dianalisis.

07

Simpan Model

Model, konfigurasi, history, dan classification report disimpan sebagai artefak.

08

Inference

Gambar eksternal diproses untuk memperoleh kelas dan confidence prediksi.
